## Instruction

> Comment installer notre projet

Cloner notre projet :

``` bash
$ git clone https://github.com/Clement-Julia/js_eval.git 
$ cd ../js_eval
```

Installer npm :
``` npm
$ npm install
```

Lancer le site en local :
``` npm
$ npm run dev
```

La commande pour les test :
``` npm
$ npm run test {le test}
```
